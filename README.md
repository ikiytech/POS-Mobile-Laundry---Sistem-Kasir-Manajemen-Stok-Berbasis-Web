🧾 POS Mobile Laundry
Sistem Kasir & Manajemen Stok Berbasis Web

✨ Fitur Utama
🛒 Kasir (Checkout Cepat)
Input produk dengan cepat, tambah/kurangi item, dan proses transaksi langsung.
📦 Manajemen Produk
Tambah, edit, dan hapus produk lengkap dengan harga, kategori, dan stok.
📊 Inventaris Stok
Monitoring stok secara real-time dan penyesuaian jumlah barang.
🧾 Laporan Transaksi
Riwayat transaksi lengkap dengan detail waktu dan total pembayaran.
⚙️ Pengaturan Toko
Kelola nama toko, alamat, nomor telepon, dan catatan struk.

⚡ Template Produk Otomatis (Auto Generate)
Fitur "Pakai Template" memungkinkan pengguna langsung membuat banyak produk sekaligus berdasarkan kategori seperti:

☕ Kopi
🛒 Sembako
🍟 Jajanan
🧺 Laundry

Saat template dipilih:

Produk otomatis dibuat
Stok langsung terisi
Siap digunakan di kasir tanpa input manual
🚀 Kelebihan
UI sederhana & mudah digunakan
Responsif (mobile & desktop)
Cocok untuk usaha laundry kecil hingga menengah
Sistem ringan & cepat
