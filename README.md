POS Mobile Laundry - Sistem Kasir & Manajemen Stok Berbasis Web

Aplikasi Point of Sale (POS) berbasis web yang dirancang khusus untuk usaha laundry, dilengkapi dengan fitur kasir, manajemen produk, stok, laporan transaksi, dan pengaturan toko dalam satu sistem yang simpel dan responsif.

✨ Fitur Utama
🛒 Kasir (Checkout Cepat)
Input produk dengan cepat, tambah/kurangi item, dan proses transaksi langsung.
📦 Manajemen Produk
Tambah, edit, dan hapus produk lengkap dengan harga, kategori, dan stok.
📊 Inventaris Stok
Monitoring stok secara real-time dan penyesuaian jumlah barang.
🧾 Laporan Transaksi
Riwayat transaksi lengkap dengan detail waktu dan total pembayaran.
⚙️ Pengaturan Toko
Kelola nama toko, alamat, nomor telepon, dan catatan struk.
🚀 Kelebihan
UI sederhana & mudah digunakan
Responsif (bisa dipakai di HP & desktop)
Cocok untuk usaha laundry kecil hingga menengah
Sistem ringan & cepat
