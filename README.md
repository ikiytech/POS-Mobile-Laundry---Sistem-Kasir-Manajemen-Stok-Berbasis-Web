# POS-Mobile-Laundry---Sistem-Kasir-Manajemen-Stok-Berbasis-Web
Aplikasi POS berbasis web untuk usaha laundry yang memudahkan transaksi, manajemen produk, stok, dan laporan dalam satu sistem. Memiliki tampilan sederhana, responsif, dan mudah digunakan, sehingga cocok untuk meningkatkan efisiensi operasional bisnis laundry kecil hingga menengah.
